# Support

OpenSyria Data Geography is the public contribution surface for Syrian administrative geography records.

## Table of Contents

- [Where to Ask](#where-to-ask)
- [Before Opening an Issue](#before-opening-an-issue)
- [Security](#security)
- [Response Expectations](#response-expectations)

## Where to Ask

Use GitHub issues in this repository for:

- incorrect geography records,
- missing approved geography records,
- source review,
- source licensing questions,
- schema proposals,
- documentation corrections.

Use `datasets-api` for:

- API endpoint behavior,
- OpenAPI or Scalar documentation issues,
- deployment, Docker, Redis, database, sync, or read-model import issues.

## Before Opening an Issue

Please include:

- the affected record ID when known,
- the affected file or dataset,
- source URLs or citations,
- source license or reuse notes when known,
- uncertainty or source conflicts,
- whether the issue came from a generated coverage report.

Do not include private, personal, restricted, military, checkpoint, surveillance, or operational-sensitive data.

## Security

Do not report security vulnerabilities or sensitive data exposure in public issues. Follow [SECURITY.md](SECURITY.md).

## Response Expectations

This project is maintained by a small maintainer team. Issues may be triaged by priority, source quality, licensing clarity, safety impact, coverage gaps, and release timing.

Maintainers may close issues that are out of scope, duplicates, unsupported by reusable sources, unsafe to publish, or better handled in another repository.
